
**Purpose**

This database is designed to facilitate the translation of 4D database applications using XLIFF (XML Localization Interchange File Format), ensuring consistent and efficient translation into various languages.

**Configuration**

The database can be opened using the latest 20.4 or 20 R5 versions.

**Features**

- Language Management: Choose a specific language for a single process or open several forms with a specific language for each window.
- Web Area Integration: When the form contains a web area, the 4D site is displayed in the selected language.
- Arabic Text Import: Arabic text can be imported by clicking the "Import Arabic Text" button.
